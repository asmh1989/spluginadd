# spluginadd



```
  自动添加插件到RNSmobiler中, 方便调试

  Options:

    -h, --help            output usage information
    -V, --version         output the version number
    -a, --add [items]     添加插件, 支持本地目录, 支持git url地址, 可以通过多次-a来添加多个
    -p, --package <name>  改变包名
    -d, --dir <name>      设定RNSmobiler目录, 默认当前目录下
    -r, --reset           重置代码
```
